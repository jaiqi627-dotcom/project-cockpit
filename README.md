# 个人项目驾驶舱

面向新产品开发的个人项目管理 PWA，覆盖八阶段项目门、行动中心、资料收件箱、技术方案论证和每日复盘。

## 数据说明

- GitHub Pages 仅托管静态网页程序。
- 项目文字数据默认保存在当前浏览器。
- 配置 Supabase 后，数据使用 AES-GCM 在浏览器内加密，再进行多设备同步。
- 照片和大型 Office 附件保存在设备本机，通过同步包、QQ 或 ToDesk 传输归档。

## 本地运行

使用任意静态文件服务器打开本目录，例如：

```bash
python3 -m http.server 4173
```

## 发布

GitHub Pages从 `main` 分支根目录发布，推送更新后网址会自动刷新。
